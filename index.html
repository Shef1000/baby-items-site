Static website

中国可访问部署建议

  1. 最稳妥：上传整个文件夹到阿里云 OSS、腾讯云 COS 或华为云 OBS 的静态网站托管。
  2. 只需要三个核心文件：index.html、styles.css、app.js；不依赖外部字体、外部 JS 或嵌入视频。
  3. 如果用中国大陆服务器/对象存储并绑定自有域名，通常需要做 ICP 备案。
  4. 课程视频不要嵌入页面，保持为普通链接；这样网站本身在中国更稳，外部平台可用性单独处理。

进度只保存在当前浏览器本地，不上传。
