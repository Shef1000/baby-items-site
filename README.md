# Baby Items Shareable Website

This is a static website with a curated list of practical, high-quality baby gear.

## Open locally

- Double-click `index.html`
- Or run a local server:

```bash
cd /Users/sheilafehring/baby-items-site
python3 -m http.server 8080
```

Then open <http://localhost:8080>.

## Share with others

Fast options:

1. Upload folder to Netlify Drop: <https://app.netlify.com/drop>
2. Push to GitHub and enable GitHub Pages.

No build step is required.
